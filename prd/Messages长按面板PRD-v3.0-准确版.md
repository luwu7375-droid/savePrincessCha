## 四、语音消息系统

### 4.1 KK "假"语音

**流程**：
```
用户点击麦克风 icon
  ↓
调用 iOS 自带语音识别
  ↓
用户说话，系统识别为文字
  ↓
弹窗显示转文字结果
  ↓
用户可编辑修改
  ↓
点击"发送"确认
  ↓
生成"假"语音气泡
  （看起来像真语音，实际是文字转的）
```

**特点**：
- 可以转文字
- 可以编辑（编辑转文字结果）
- 可以播放（用 TTS 播放转文字结果）
- 无法播放原始音频
- Cha 知道这是"假"语音

### 4.2 Cha "真"语音

**流程**：
```
Cha 生成回复（文字）
  ↓
添加自然感 prompt（停顿、语气等）
  ↓
调用 TTS 生成音频
  ↓
打包成语音气泡发送
```

**特点**：
- 可以播放真实音频
- 可以转文字
- 可以重新生成
- 可以调整倍速
- KK 可以听完整语音

### 4.3 语音气泡样式

详见 HTML 设计文件

## 五、新增功能详解

### 5.1 删除

**仅本地删除**

```
前：
  KK: 你怎么看？
  Cha: 我觉得很不错...

后：
  KK: 你怎么看？
  （Cha 消息消失）
```

**数据处理**：
- 前端：消息消失，下一条上移
- 后端：标记 `is_deleted = true`（不真正删除）
- 对方视角：知道有消息被删除

### 5.2 撤回

**仅 KK 消息支持**

```
前：
  KK: 你怎么看？

后：
  你撤回了一条消息
  原文：你怎么看？
```

**数据处理**：
- 前端：显示撤回提示 + 原文
- 后端：标记 `is_recalled = true`，保存 `recalled_original`
- Cha 视角：能看到撤回的原文

### 5.3 收藏

**所有消息都支持**

```
长按消息 → 点击[收藏] → 显示"已收藏"
```

**收藏夹**：
- 快速筛选：`[最近使用]  [图片与视频]  [聊天记录]  [语音]`
- 按收藏时间倒序
- 最近使用 = 近 3 天有 2 次以上打开

### 5.4 多选

**视觉**：
```
☑️ [消息气泡]  ← 左侧蓝色勾选框
☐ [消息气泡]  ← 未选中灰色空心
```

**流程**：
```
长按消息 → 点击[多选]
  ↓
进入多选模式
  ↓
顶部显示：[取消] 已选中 n 条 [全选]
  ↓
底部 bar：[逐条转发]  [合并转发]  [收藏]  [删除]
```

### 5.5 朗读

**仅 Cha 文字消息支持**

```
长按 Cha 文字 → 点击[朗读]
  ↓
长按面板关闭
  ↓
TTS 播放文字
  ↓
可暂停、继续
```

### 5.6 转发

**支持范围**：
- 其他 chat 内联系人
- Cha Phone 内 App

**转发方式**：
- 逐条转发：一条一条转
- 合并转发：生成卡片转发多条

### 5.7 转文字（语音）

**流程**：
```
长按语音 → 点击[转文字]
  ↓
调用语音转文字 API
  ↓
显示转文字结果
```

### 5.8 倍速（语音）

**仅 Cha 语音支持**

```
长按 Cha 语音 → 点击[倍速]
  ↓
选择：1.0x / 1.5x / 2.0x
  ↓
按选定倍速播放
```

### 5.9 编辑描述（图片）

**仅 KK 图片支持**

```
长按图片 → 点击[编辑]
  ↓
弹窗输入描述
  ↓
保存显示在图片下方
```

### 5.10 查看描述（图片）

**仅 Cha 图片支持**

```
长按图片 → 点击[查看描述]
  ↓
弹窗显示生成提示词
  ↓
可复制提示词
```

## 六、转发功能详解

### 6.1 单条转发（逐条转发）

```
长按消息 → 点击[转发]
  ↓
选择转发目标：
  • 其他 chat 联系人
  • Cha Phone Notes
  • Cha Phone Diary
  • Cha Phone Photos
  ↓
转发到该目标
```

### 6.2 多选转发

```
长按 → [多选] → 选中多条
  ↓
底部 bar → [逐条转发]
  ↓
选择转发目标
  ↓
所有选中消息逐条转发到目标
```

### 6.3 合并转发

```
长按 → [多选] → 选中多条
  ↓
底部 bar → [合并转发]
  ↓
生成卡片：
  ┌──────────────────────┐
  │ KK和Cha的聊天记录    │
  │ 2024-06-24 14:00     │
  ├──────────────────────┤
  │ 👤 KK                │
  │ 你怎么想？           │
  │                      │
  │ 👤 Cha               │
  │ 我觉得很不错...      │
  │                      │
  │ 👤 KK                │
  │ [图片信息]           │
  └──────────────────────┘
  ↓
选择转发目标
  ↓
卡片转发到目标
```

## 七、数据模型

### Message 表新增字段

```
// 删除和撤回
is_deleted: boolean
deleted_at: timestamp

is_recalled: boolean
recalled_at: timestamp
recalled_original: string

// 编辑历史
edited: boolean
edit_history: [{ version, content, edited_at }]

// 收藏
is_favorited: boolean
favorited_at: timestamp
favorite_access_count: number
favorite_last_accessed: timestamp

// 语音
audio_url: string
audio_duration: number
audio_type: "real" | "fake"
audio_transcribed_text: string

// 图片
image_description: string
image_prompt: string
```
