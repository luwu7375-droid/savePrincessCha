    Personal Memories长期记忆
    已启用 always inject
    persona_memories
    每轮按分类注入（L1）。persona / life / relation 域覆盖人格、生活、关系上下文。通过"记忆管理（高级）"手动管理。
    Core Profile常驻画像
    已启用 always inject
    mastodon_profile —
    每轮注入，用于理解用户。参考资料，不复述来源。
    Project Memory项目记忆
    已启用 route inject
    project_memory
    话题路由为 project_work 时注入（L2）。work / writing 域，涵盖项目进度、代码问题、写作 OC 等内容。
    Timeline Archive时间线档案
    已启用 retrieval only
    mastodon_timeline
    只在年份、地点、事件、经历类问题时按需召回。
    OpenAI Archive前世档案
    已启用 retrieval only
    openai_archive
    原始 OpenAI 导出不会在运行时直接读取；这里显示的是已整理入库的历史档案按需召回状态。
    Conversation History历史对话
    已启用 retrieval only
    conversation_history
    触发词激活，跨会话语义检索，top-5 召回（L3）。
    Auto Memory Vault自动记忆库
    已启用
    auto_memory_vault
    对话后自动提取候选，符合白名单（project / fact / preference）的条目沉淀到 memories 表，经 L1/L2 注入后续回复。可在最近更新区域查看并禁用/删除。
    旧记忆工具
    Legacy Manual Memory 管理工具
    memories 表 手动记忆条目，经 L1/L2 注入回复
    memory_buckets 表 旧沉淀摘要桶，不注入回复
    distill candidates 从对话提取，待确认，不注入回复
    memories 表条目通过 persona_memories（L1）和 project_memory（L2）参与回复注入。memory_buckets 和 distill candidates 属于旧工具，仅用于管理和迁移。
